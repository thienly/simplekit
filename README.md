## Scope of works
- In order to support customer shopping online. a CLI appplication is created. The architecture of application would support:  
- High availability   
- Scalability  
- Maintainability 
## Admin scope
>- Login to admin
>- Add new products to store 
## Customers:
>- Login to system
>- Browse
>- Add Product to cart
>- Check out cart
>- Deposit money to account to increase balance
